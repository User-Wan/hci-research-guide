# 李克特量表（Likert Scale）

## 概述

**李克特量表**是一种心理测量量表，用于测量态度、意见和感知强度。它由 Rensis Likert 于 1932 年开发，是问卷调查中最常用的量表类型。

---

## 量表类型

### 5 点量表

```
非常不同意 | 不同意 | 中立 | 同意 | 非常同意
    1        2       3      4        5
```

**适用场景**：
- 测量态度和意见
- 评估满意度
- 收集偏好数据

### 7 点量表

```
非常不同意 | 不同意 | 有点不同意 | 中立 | 有点同意 | 同意 | 非常同意
    1        2          3         4        5       6        7
```

**适用场景**：
- 需要更细粒度
- 测量感知强度
- 区分度要求高

### 4 点量表（无中立选项）

```
非常不同意 | 不同意 | 同意 | 非常同意
    1        2       3        4
```

**适用场景**：
- 强迫选择
- 减少中立偏见
- 测量明确态度

## 使用方法

### 步骤 1：确定测量目标

**明确你要测量什么**：
- 态度（Attitude）
- 意见（Opinion）
- 满意度（Satisfaction）
- 感知（Perception）

### 步骤 2：设计问题

**问题类型**：
- **陈述句**："我认为这个系统很容易使用"
- **问题句**："你对这个系统的满意度如何？"

**设计原则**：
- 使用清晰、简洁的语言
- 避免双重否定
- 避免引导性语言
- 正向和反向问题混合使用

### 步骤 3：选择量表类型

**决策树**：

```
需要中立选项？
├── 是 → 5 点或 7 点量表
└── 否 → 4 点量表

需要细粒度？
├── 是 → 7 点量表
└── 否 → 5 点量表
```

### 步骤 4：数据分析

**描述统计**：
- 均值（Mean）
- 标准差（Standard Deviation）
- 频率分布

**推断统计**：
- t 检验（比较两组）
- ANOVA（比较多组）
- 相关分析（变量关系）

## 问卷示例

### 示例 1：系统满意度

```
请根据你使用系统的体验，对以下陈述进行评分：

1. 我对这个系统的整体满意度很高
   非常不同意 ①  ②  ③  ④  ⑤ 非常同意

2. 这个系统满足了我的需求
   非常不同意 ①  ②  ③  ④  ⑤ 非常同意

3. 我会推荐这个系统给其他人
   非常不同意 ①  ②  ③  ④  ⑤ 非常同意

4. 这个系统的响应速度很快
   非常不同意 ①  ②  ③  ④  ⑤ 非常同意

5. 我对这个系统的功能很满意
   非常不同意 ①  ②  ③  ④  ⑤ 非常同意
```

### 示例 2：易用性评估

```
请根据你使用系统的体验，对以下陈述进行评分：

1. 学习使用这个系统很容易
   非常不同意 ①  ②  ③  ④  ⑤ 非常同意

2. 我能够快速完成任务
   非常不同意 ①  ②  ③  ④  ⑤ 非常同意

3. 系统的界面设计直观
   非常不同意 ①  ②  ③  ④  ⑤ 非常同意

   非常不同意 ①  ②  ③  ④  ⑤ 非常同意

5. 系统的错误提示清晰易懂
   非常不同意 ①  ②  ③  ④  ⑤ 非常同意
```

## 计分方法

### 正向问题

直接使用原始分数：
- 非常不同意 = 1
- 不同意 = 2
- 中立 = 3
- 同意 = 4
- 非常同意 = 5

### 反向问题

使用反向计分：
- 非常不同意 = 5
- 不同意 = 4
- 中立 = 3
- 同意 = 2
- 非常同意 = 1

### 计算总分或均值

**总分** = 所有题目的分数之和

**均值** = 总分 / 题目数量

---

## 数据分析

### 描述统计

| 指标 | 用途 | 计算方法 |
|------|------|---------|
| **均值** | 数据的中心位置 | 所有值的平均 |
| **中位数** | 数据的中心位置（不受极端值影响） | 排序后的中间值 |
| **标准差** | 数据的离散程度 | 方差的平方根 |
| **频率** | 各选项的选择次数 | 计数 |

### 推断统计

| 方法 | 用途 | 适用条件 |
|------|------|---------|
| **t 检验** | 比较两组均值 | 两组数据 |
| **ANOVA** | 比较多组均值 | 三组及以上 |
| **相关分析** | 分析变量关系 | 两个连续变量 |
| **回归分析** | 预测因变量 | 一个或多个自变量 |

### 效应量

计算 Cohen's d 或 η² 来衡量效应大小。

## 常见问题

### Q1：应该用 5 点还是 7 点量表？

**建议**：
- 大多数情况使用 **5 点量表**
- 需要更细粒度时使用 **7 点量表**
- 4 点量表用于强迫选择

### Q2：是否应该提供中立选项？

**建议**：
- 提供中立选项可以减少强迫选择的压力
- 但可能导致中立偏见
- 根据研究目标决定

### Q3：如何处理缺失值？

**建议**：
- 删除缺失值（如果缺失较少）
- 使用均值替换（如果缺失较多）
- 使用多重插补（如果缺失很多）

### Q4：如何确保量表的信效度？

**建议**：
- 使用经过验证的量表
- 进行预测试
- 计算 Cronbach's α（信度）
- 进行因子分析（效度）

## 重点总结

- 李克特量表是**最常用的态度测量工具**
- **5 点量表**适合大多数情况
- **7 点量表**提供更细粒度
- **正向和反向问题**混合使用减少偏见
- 使用**均值、标准差**进行描述统计
- 使用 **t 检验、ANOVA** 进行推断统计
