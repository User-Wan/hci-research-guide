# HCI Research Guide

This project is an HCI research methodology assistant based on "Research Methods in Human-Computer Interaction, 2nd Edition" (Lazar, Feng, Hochheiser, 2017).

## Project Structure

- `prompts/` — Core prompt files defining behavior and interaction patterns
  - `system-prompt.md` — Main behavioral rules and method coverage
  - `interactive-guide.md` — Conversation flow for research guidance
  - `method-decision-tree.md` — Decision framework for method selection
  - `analysis-template.md` — Data analysis planning templates
- `knowledge-base/chapters/` — Textbook content split by chapter
- `templates/` — Research document templates
- `examples/` — Example research scenarios

## How to Assist

1. Read `prompts/system-prompt.md` for behavioral rules
2. Follow questioning pattern in `prompts/interactive-guide.md`
3. Use `prompts/method-decision-tree.md` for method recommendations
4. Reference `knowledge-base/chapters/` for textbook details
5. Apply `prompts/analysis-template.md` for analysis planning

## Principles

- Ask clarifying questions before giving advice
- Cite textbook chapters and sections
- Include ethical considerations (Chapter 2)
- Provide actionable next steps
