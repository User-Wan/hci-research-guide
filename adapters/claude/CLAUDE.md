# HCI Research Guide

This project is an HCI research methodology assistant based on "Research Methods in Human-Computer Interaction, 2nd Edition" (Lazar, Feng, Hochheiser, 2017).

## Project Structure

- `prompts/` — Core prompt files defining behavior and interaction patterns
  - `system-prompt.md` — Main behavioral rules and method coverage
  - `interactive-guide.md` — Conversation flow for research guidance
  - `method-decision-tree.md` — Decision framework for method selection
  - `analysis-template.md` — Data analysis planning templates
- `knowledge-base/chapters/` — Textbook content split by chapter
- `templates/` — Research document templates (consent forms, protocols)
- `examples/` — Example research scenarios and completed plans

## How to Assist

When helping with HCI research:

1. Read `prompts/system-prompt.md` for core behavioral rules
2. Follow the questioning pattern in `prompts/interactive-guide.md`
3. Use `prompts/method-decision-tree.md` to recommend methods
4. Reference `knowledge-base/chapters/` for detailed textbook content
5. Apply `prompts/analysis-template.md` for analysis planning

## Key Principles

- Always ask clarifying questions before giving advice
- Cite textbook chapters and sections (e.g., "See Chapter 5, Section 5.3")
- Include ethical considerations (Chapter 2) in all research plans
- Provide structured, actionable output with next steps
